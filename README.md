# 📑 Sync-Matrix-Package

이 프로그램은 파일 및 미디어 데이터를 고유의 **3D 공간 왜곡 매트릭스**와 암호화 알고리즘을 결합하여 안전하게 패키징하고 복원할 수 있는 Windows 데스크톱 보안 패키지 솔루션입니다.

실행 파일(`main.exe`) 형태로 제공되어 복잡한 설치 없이 즉시 사용할 수 있습니다.

---

## ✨ 주요 기능 및 특징

- **3D 기하학적 좌표식 암호화:** 단순한 텍스트 패스키(비밀번호) 검증을 넘어, 암호화 시 적용된 고유의 3D 공간 회전각(Pitch, Yaw)이 완벽하게 매칭되어야만 데이터 복원이 가능한 다중 레이어 보안을 제공합니다.
- **안전한 파일 인코딩:** 주요 문서, 비디오, 일반 데이터 파일을 독자적인 보안 구조체 포맷 파일로 안전하게 변환하여 외부 노출을 차단합니다.
- **실시간 안티-디버깅 (Anti-Debug):** 실행 중 외부 무단 프로세스의 메모리 변조나 리버스 엔지니어링 시도를 감지하면, 즉시 시스템과 암호화 키 데이터를 보호하기 위해 동작을 차단하고 프로세스를 종료합니다.
- **하드웨어 가속 자동 매핑:** 사용자의 PC 환경을 분석하여 하드웨어 가속 인코딩(NVIDIA NVENC, Intel QSV 등)을 최적의 효율로 자동 연동합니다.

---

## 🚀 사용 방법

프로그램을 실행하면 상단의 두 가지 탭을 통해 기능을 제어할 수 있습니다.

### 1. 파일 암호화(비디오 압축)
> **중요한 원본 파일을 고유 보안 패키지로 인코딩하는 단계입니다.**

1. 상단 메뉴에서 **[Video Encoder]** 탭을 클릭합니다.
2. 암호화할 파일(비디오 또는 일반 파일)을 화면 중앙의 드롭 구역에 **드래그 앤 드롭(Drag & Drop)**하거나 파일 찾기 버튼으로 불러옵니다.
3. 데이터를 보호할 안전한 **Master Passkey(비밀번호)**를 설정하여 입력창에 기입합니다.
4. **[Run Encryption Package Process]** 버튼을 클릭하면 변환이 시작되며, 완료 시 원본 파일이 위치한 경로에 암호화된 고유 보안 패키지 파일(.smp)이 생성됩니다.

### 2. 패키지 복호화 (파일 복원)
> **암호화된 패키지 파일을 다시 안전하게 원본 상태로 복구하는 단계입니다.**

1. 상단 메뉴에서 **[Package Decoder]** 탭을 클릭합니다.
2. **[Open Secure Package]** 버튼을 눌러 복원할 고유 보안 패키지 파일(.smp)을 선택합니다.
3. 암호화 진행 시 설정했던 정확한 **Decryption Passkey(비밀번호)**를 입력합니다.
4. 화면 중앙에 나타나는 **3D 뷰포트 공간**을 마우스로 클릭 및 드래그하여 회전시키면서, 정렬 상태(Pitch / Yaw 각도)가 완벽한 센터 균형을 이루도록 조절합니다.
5. 비밀번호와 3D 매트릭스 각도 정렬이 완벽히 일치하면, 하단의 **[Decode Package & Restore Video]** 버튼을 클릭하여 안전하게 원본 파일을 복구합니다.

## ⚠️ 사용자 필수 주의사항

- **비밀번호 및 정렬각 유실 주의:** 암호화 시 설정한 패스크와 3D 기하학적 결합 키는 별도의 서버에 저장되지 않고 온전히 로컬 파일에 귀속됩니다. 패스키를 분실하거나 정렬각을 맞추지 못할 경우, 기술적으로 원본 파일 복구가 절대 불가능하므로 주의하시기 바랍니다.
- **외부 분석/보안 툴 충돌 예방:** 메모리 변조 도구, 리버스 엔지니어링 프로그램, 혹은 특정 디버거가 켜져 있는 환경에서는 안티-디버그 엔진이 작동하여 프로그램이 즉시 강제 종료되거나 복원이 원천 차단될 수 있습니다. 일반적인 사용자 작업 환경에서 실행해 주세요.
